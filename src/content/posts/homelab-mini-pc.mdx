## Einleitung
Self‑Hosting boomt – nicht nur aus Datenschutzgründen, sondern auch, weil **Mini‑PCs** heute leise, sparsam und erstaunlich leistungsfähig sind. Dieser Guide zeigt dir ein **pragmatisches Setup**: Docker als Laufzeit, Traefik als Reverse‑Proxy mit TLS, Portainer fürs GUI – plus Security‑Basics und sinnvolle Dienste für den Alltag.

### Für wen ist dieser Guide?
- Du willst in *unter einer Stunde* ein stabiles Homelab.  
- Du bevorzugst **Wartbarkeit** statt Bastelorgien.  
- Du möchtest deine **Dienste per Domain + HTTPS** erreichen.

## Basis-OS installieren
Ich empfehle **Ubuntu Server LTS** oder **Debian** (minimal). Nach der Installation:

```bash
sudo apt update && sudo apt upgrade -y
sudo timedatectl set-timezone Europe/Berlin
```

## Docker & Compose
Docker bequem per Script installieren und den User zur Gruppe hinzufügen:

```bash
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER
newgrp docker
```

## Traefik als Reverse Proxy (TLS via Let's Encrypt)
Traefik nimmt dir Zertifikate, Routing und einfache Observability ab.

```yaml
# docker-compose.yml (Grundlage)
services:
  traefik:
    image: traefik:v3.1
    command:
      - "--providers.docker=true"
      - "--entrypoints.web.address=:80"
      - "--entrypoints.websecure.address=:443"
      - "--certificatesresolvers.le.acme.httpchallenge=true"
      - "--certificatesresolvers.le.acme.email=you@example.com"
      - "--certificatesresolvers.le.acme.storage=/letsencrypt/acme.json"
    ports:
      - "80:80"
      - "443:443"
    volumes:
      - /var/run/docker.sock:/var/run/docker.sock:ro
      - ./letsencrypt:/letsencrypt
    restart: always
```

Für **jeden Dienst** definierst du Labels, z. B. für Portainer:

```yaml
  portainer:
    image: portainer/portainer-ce:latest
    ports: ["9443:9443"]
    volumes:
      - /var/run/docker.sock:/var/run/docker.sock
      - portainer_data:/data
    restart: always
    labels:
      - "traefik.enable=true"
      - "traefik.http.routers.portainer.rule=Host(`portainer.deinedomain.de`)"
      - "traefik.http.routers.portainer.entrypoints=websecure"
      - "traefik.http.routers.portainer.tls.certresolver=le"
```

## Sicherheit & Updates
- **Unattended‑Upgrades** für kritische Patches aktivieren.  
- **SSH Hardening** (Key‑Auth, Fail2ban).  
- **Backups**: `restic` + S3/Backblaze, Rotation 7‑14‑30.

## Monitoring light
- **Uptime‑Kuma** für Reachability  
- **Glances** oder **Netdata** lokal

## Fazit
Mit Docker, Traefik und Portainer hast du ein **pflegeleichtes, skalierbares** Setup. Starte klein, automatisiere Backups – und erweitere schrittweise um Dienste, die dir echten Mehrwert bringen.
